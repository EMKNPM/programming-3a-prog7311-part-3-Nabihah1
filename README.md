# TechMove Logistics Management System

## Overview

The TechMove Logistics Management System is a web-based application developed using ASP.NET Core MVC and ASP.NET Core Web API. The system allows users to manage clients, contracts, and service requests through a user-friendly interface while providing RESTful API endpoints for data management and integration.

## Features

* Client Management (Create, Read, Update, Delete)
* Contract Management
* Service Request Management
* RESTful Web API Endpoints
* Swagger API Documentation
* Docker Container Support
* Integration Testing

## Technologies Used

* ASP.NET Core MVC
* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* Docker
* Swagger/OpenAPI
* xUnit Integration Testing

## Docker

The application can be run using Docker containers.
